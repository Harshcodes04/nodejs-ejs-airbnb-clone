# nodejs-ejs-airbnb-clone

A mini Airbnb-style web application built using Node.js, Express, and EJS.  
Users can add homes with details like price, rating, address, and photo URL, and view them dynamically on the homepage.

## Features

- Add homes with price, rating, address, and photo URL
- Dynamic rendering using EJS
- Express routing and form handling
- Tailwind CSS based UI

## Tech Stack

- Node.js
- Express.js
- EJS
- Tailwind CSS

## How to Run

```bash
npm install
npm start
```
